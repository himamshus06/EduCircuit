What is Python?





Python is a high-level, versatile programming language that is widely recognized for its simplicity and readability. It was created by Guido van Rossum and first released in 1991. Python is often chosen by beginners due to its intuitive syntax, which closely resembles natural language, making it easier to learn and use compared to other programming languages. At the same time, it is powerful enough to handle complex applications, making it a favourite among experienced developers.

One of Python's standout features is its versatility. It is used in a wide range of fields, including web development, data science, artificial intelligence, machine learning, automation, and scientific computing. Python supports multiple programming paradigms such as procedural, object-oriented, and functional programming, which allows developers to solve problems in various ways.

Python is also an open-source language, meaning it is free to use and has a massive community contributing to its development. Its extensive standard library and third-party packages, such as NumPy, pandas, and TensorFlow, make it an ideal choice for diverse projects. Additionally, Python is cross-platform, meaning code written in Python can run on different operating systems like Windows, macOS, and Linux. Overall, Python is a beginner-friendly, efficient, and flexible programming language that is widely used in industries around the world. Whether you're building a simple script or developing a complex application, Python provides the tools and support needed to succeed.



Basic Syntaxes:





1. Print Output:

Display text or values.

print("Hello, world!")

2. Variables:

Store data without declaring types.

x = 5
y = "hehehehe"

print(x, y)

3. Data Types:

integer_var = 10
float_var = 3.14
string_var = "Hello"
boolean_var = True

4. Input:

Take user input.

name = input("What is your name? ")
print("Hello, " + name)

5. Conditional Statements:

Make decisions with if, elif, else.

x = 10
if x > 5:
  print("x is greater than 5")
else:
  print("x is 5 or less")

6. Loops:

Repeat actions.

I. For loop:

Repeat actions.
for i in range(5):
  print(i)

II. While loop:

Repeats while a condition is true.

count = 0
while count < 5:
  print(count)
  count += 1

7. Functions:

Reusable blocks of code.

def greet(name):
  return "Hello, " + name

print(greet("hehehehe"))



Conditionals





Conditionals allow decisions based on conditions.

Syntax

if condition:
  # Code to execute if the condition is True
elif another_condition:
  # Code to execute if the previous condition is False
else:
  # Code to execute if no conditions are True



Example

x = 10
if x > 5:
  print("x is greater than 5")
elif x == 5:
  print("x is equal to 5")
else:
  print("x is less than 5")





Loops

1. For Loop -

Iterates over a sequence (like a range, list, or string).

Syntax:

for variable in sequence:

  # Code to execute in each iteration

Example:

  print(i)

2.While Loop -

Repeats as long as a condition is true.

Syntax:

while condition:

  # Code to execute

Example:

while count < 5:

  print(count)

Type Conversion





Type conversion in Python refers to converting one data type to another. It can be either implicit (automatic) or explicit (manual). Type conversion in Python refers to converting one data type to another. It can be either implicit (automatic) or explicit (manual). Here's a concise overview:



1. Implicit Type Conversion

Python automatically converts one type to another
(e.g., integer to float).

Example:

x = 5 # Integer

y = 2.5 # Float

z = x + y # x is implicitly converted to float

print(z)


2. Explicit Type Conversion

You manually convert using Python's built-in functions.

Common Functions:

o int() - Converts to integer.

o float() - Converts to float.

o str() - Converts to string.

o list() - Converts to a list.

o tuple() - Converts to a tuple.



Examples:

# Integer to String

x = 5

x_str = str(x)

print(x_str, type(x_str))

# String to Integer

y = "10"

y_int = int(y)

print(y_int, type(y_int))

# Float to Integer

z = 3.14

z_int = int(z)

print(z_int)



3. Conversion in Input

Input is always a string; convert it to other types as needed.

age = int(input("Enter your age: ")) # Convert to integer

print(age, type(age))

Type conversion helps ensure the correct data types for
operations.



Operators







1. Arithmetic Operators

Used for mathematical calculations.

+   # Addition

-   # Subtraction

*   # Multiplication

/   # Division (returns float)

//   # Floor Division (returns integer part of division)

%   # Modulus (remainder)

**   # Exponentiation (power)





Example:

x = 10

y = 3

print(x + y)   # 13

print(x / y)  # 3.333...

print(x % y)   # 1

print(x ** y)  # 1000



2. Comparison Operators

Compare two values; return True or False.

==   # Equal to

!=   # Not equal to

>   # Greater than

<   # Less than

>=   # Greater than or equal to

<=   # Less than or equal to



Example:

x = 5

y = 10

print(x == y)   # False

print(x < y)   # True



3. Logical Operators

Combine conditional statements.

and   # True if both conditions are True

or   # True if at least one condition is True

not   # Reverses the condition



Example:

x = 5

y = 10

print(x > 0 and y > 0)   # True

print(x > 0 or y < 0)   # True

print(not(x > 0))   # False



4. Assignment Operators

Assign values with shorthand operations.

=   # Assign

+=   # Add and assign

-=   # Subtract and assign

*=   # Multiply and assign

/=   # Divide and assign



Example:

x = 5

x += 3   # x = x + 3

print(x)   # 8



5. Membership Operators

Check for membership in sequences (e.g., lists, strings).

in  # True if value exists

not in  # True if value does not exist



Example:

name = "hehehehehe"

print("h" in name)   # True

print("z" not in name)   # True



Absolute Value





The absolute value of a number is its non-negative value,
regardless of its sign. In Python, you can find the
absolute value using the built-in function abs().

++++

Syntax:

abs(number)

Examples



1. Absolute value of an integer:

print(abs(-10)) # Output: 10

print(abs(10)) # Output: 10



2. Absolute value of a float:

print(abs(-3.14)) # Output: 3.14

print(abs(3.14)) # Output: 3.14



3. Absolute value in expressions:

x = -15 + 5

print(abs(x)) # Output: 10

The abs() function is simple and works for both integers
and floating-point numbers.



Absolute value in Math Function

To compute the absolute value using Python's
math module, you can use math.fabs(), which
works specifically with floats. It's part
of the math module, so you need to import it first.

Syntax

import math

math.fabs(number)



Examples



1. Using abs():

print(abs(-10)) # Output: 10

print(abs(-3.14)) # Output: 3.14

print(abs(3 + 4j)) # Output: 5.0 (Magnitude of a complex number)



2. Using math.fabs():

import math

print(math.fabs(-10)) # Output: 10.0

print(math.fabs(-3.14)) # Output: 3.14

# print(math.fabs(3 + 4j)) # Error: Only works with floats or integers
