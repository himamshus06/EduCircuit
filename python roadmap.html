What is Python?





Python is a high-level, versatile programming language that is widely recognized for its simplicity and readability. It was created by Guido van Rossum and first released in 1991. Python is often chosen by beginners due to its intuitive syntax, which closely resembles natural language, making it easier to learn and use compared to other programming languages. At the same time, it is powerful enough to handle complex applications, making it a favourite among experienced developers.

One of Python's standout features is its versatility. It is used in a wide range of fields, including web development, data science, artificial intelligence, machine learning, automation, and scientific computing. Python supports multiple programming paradigms such as procedural, object-oriented, and functional programming, which allows developers to solve problems in various ways.

Python is also an open-source language, meaning it is free to use and has a massive community contributing to its development. Its extensive standard library and third-party packages, such as NumPy, pandas, and TensorFlow, make it an ideal choice for diverse projects. Additionally, Python is cross-platform, meaning code written in Python can run on different operating systems like Windows, macOS, and Linux. Overall, Python is a beginner-friendly, efficient, and flexible programming language that is widely used in industries around the world. Whether you're building a simple script or developing a complex application, Python provides the tools and support needed to succeed.



Basic Syntaxes:





1. Print Output:

Display text or values.

print("Hello, world!")

2. Variables:

Store data without declaring types.

x = 5
y = "hehehehe"

print(x, y)

3. Data Types:

integer_var = 10
float_var = 3.14
string_var = "Hello"
boolean_var = True

4. Input:

Take user input.

name = input("What is your name? ")
print("Hello, " + name)

5. Conditional Statements:

Make decisions with if, elif, else.

x = 10
if x > 5:
  print("x is greater than 5")
else:
  print("x is 5 or less")

6. Loops:

Repeat actions.

I. For loop:

Repeat actions.
for i in range(5):
  print(i)

II. While loop:

Repeats while a condition is true.

count = 0
while count < 5:
  print(count)
  count += 1

7. Functions:

Reusable blocks of code.

def greet(name):
  return "Hello, " + name

print(greet("hehehehe"))



Conditionals





Conditionals allow decisions based on conditions.

Syntax

if condition:
  # Code to execute if the condition is True
elif another_condition:
  # Code to execute if the previous condition is False
else:
  # Code to execute if no conditions are True



Example

x = 10
if x > 5:
  print("x is greater than 5")
elif x == 5:
  print("x is equal to 5")
else:
  print("x is less than 5")
